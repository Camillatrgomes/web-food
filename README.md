## Sobre o Web Food

Este projeto foi desenvolvido com o objetivo de aprimorar meus conhecimentos em programação, utilizando **HTML**, **CSS com Bootstrap** e **JavaScript puro**.  
O tema **“food”** foi escolhido para simular um site de pedidos de comida, permitindo explorar componentes e interações essenciais desse tipo de aplicação.

Durante o desenvolvimento, foram implementadas as seguintes funcionalidades:

- Alternância entre **tema claro e escuro**
- **Contagem regressiva** em tempo real
- Integração com a **API ViaCEP** para preenchimento automático de endereço
- Projeto totalmente responsivo para mobile

O projeto serviu como prática para aprofundar o uso do JavaScript na manipulação do DOM, no consumo de APIs externas e na criação de interfaces mais dinâmicas e funcionais.
